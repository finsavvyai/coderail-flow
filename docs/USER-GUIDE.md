# User Guide

## Getting Started

### Quick Start

1. **Sign Up**
   - Visit https://coderail-flow.pages.dev
   - Create your account
   - Verify your email

2. **Create Your First Flow**
   - Navigate to Dashboard
   - Click "New Flow"
   - Choose a template or start from scratch

3. **Record Your Flow**
   - Click "Start Recording"
   - Perform actions in your browser
   - Click "Stop Recording"

4. **Execute Your Flow**
   - Click "Run Flow"
   - Watch it execute in real-time

## Features

### Flow Recording
- **Browser Extension**: Install from Chrome Web Store
- **Template Library**: Start with pre-built templates
- **Custom Steps**: Add navigation, clicks, assertions

### Execution
- **Real-time Preview**: Watch your flow execute live
- **Screenshot Capture**: Automatic screenshots
- **Video Recording**: Full video of execution
- **Subtitle Generation**: Auto-generated captions

### Scheduling
- **Cron Scheduling**: Run flows on a schedule
- **Event Triggers**: Trigger via webhooks
- **Retry Logic**: Automatic retry on failure
